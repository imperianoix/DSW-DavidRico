Este sitio web ha sido creado con el propósito de promover el conocimiento sobre el HTML, nos enfocaremos principalmente en el aprendizaje de Flex Box y de Listas y enlaces.

Flex Box

Es un modelo de diseño en CSS, es una herramienta importante a la hora de usar html. Flex box viene de la palabra en inglés Flexible Box que significa caja flexible. Dependiendo de las necesidades del diseño está caja va a cambiar, su altura y anchura va a cambiar. Se puede decir que Flex box es el primer sistema de diseño enfocado a la creación de interfaces web.

Listas y enlaces

Usar listas se compararia con usar numeración y viñetas en word con el uso de etiquetas como ol, ul, entre otras. Usamos la etiqueta ol para realizar una lista ordenada por ejemplo, y para hacer una lista desordenada usamos ul. La etiqueta para abrir esos elementos en la lista sea ordenada o desordenada es li. La etiqueta más importante del html es la etiqueta enlaces, que nos permite dar clic en un texto que se ha configurado como hipervínculo y nos puede llevar a otra página web.
